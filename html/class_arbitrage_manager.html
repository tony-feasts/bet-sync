void ArbitrageManager::parseOddsData ( const nlohmann::json & data)
private

Parses odds data into the odds map.

The odds map is structured as follows:

  • Outer key: Match identifier (e.g., "Team A vs Team B").
  • Inner key: Outcome name (e.g., "Team A wins").
  • Value: OddsData structure containing bookmaker, outcome name, commencement time, and odds value.
Parameters
dataThe JSON data containing match and odds information.
